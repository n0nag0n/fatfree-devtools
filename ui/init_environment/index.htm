Welcome to the Fat-Free DevTools Setup!

Fat-Free Framework is a loosely opinionated framework for PHP. You can configure your project in an unlimited amount of possibilities because of how simple the framework is itself.

Take the below code for example:

<?php

require 'vendor/autoload.php';

$f3 = \Base::instance();
$f3->route('GET /', function() {
	echo 'Hello, world!';
});
$f3->run();

The above code is complete to render the root route (/) to echo out a text response. Pretty simple. But the more complex your application becomes, the more difficult it is to manage all its parts.

The goal of Fat-Free DevTools is to help create projects in a slightly more opinionated manner. YOU DO NOT HAVE TO FOLLOW EVERYTHING IN THIS SETUP FOR YOUR PROJECT! You can simple start your project if you want, and then create your own controllers, routes, config files, whatever you want!

With all this said, let's go ahead and get the environment setup.
